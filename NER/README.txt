This is the folder for the NER part of this project.

MeSh-2021-XML.ipynb contains the code for extracting chemical terms from MeSH suppl2021.xml (which contains the chemcial terms). Due to size limit, supp2021.xml is not included but it is available at this website https://www.nlm.nih.gov/databases/download/mesh.html.

Acronyms-from-supp.ipynb contains the code for extracting acronyms from MeSH (supp_names_ui_refer_ui.csv) using different criteria.

DA-1030.ipynb contains the code for doing data augmentation and the roudntripping test.

Capstone project report is included as Capstone Project.pdf which is a 16-page document.
